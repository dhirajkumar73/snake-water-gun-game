# Snake Water Gun Game 🐍💧🔫

This is a simple Python-based Snake Water Gun game created using the `random` module.

## 📌 Features

* User vs Computer gameplay
* Random computer choices
* Simple game logic
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run:

   ```
   python main.py
   ```
